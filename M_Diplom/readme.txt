основной язык: js 
js фреймворки: vue, webpack, bluebird (js Promises) 
платформа в которой все крутится: electron (chromium) https://electronjs.org 
библиотека по работе с VK API: https://github.com/negezor/vk-io

использованные методы апи: 
https://vk.com/dev/database.getCountries - найти города 
https://vk.com/dev/database.getCities - найти города 
https://vk.com/dev/groups.search - поиск в группах 
https://vk.com/dev/groups.getMembers - получить юзеров из группы с дополнительными параметрами, такмим как sex - пол 

Так же был испольщован метод execute из VK API. Данный метод позволяет, на так называемом vkscript, написать который может обращатся к апи. 
Этот код будет запущен на стороне VK и вернет необходимые данные. 
Этот метод удобный так как за один такой запрос можно сделать 25 запросов к VK API. 
В то время как для выполненеия единичныз запросов - необходимо выждать 3 секунды между запросами 

Алгоритм: 
1) выбрать страну 
2) выбрать город 
3) ввести запрос 
2) ждать 

По коду: 
1) вызывает методл для получения стран 
2) вызывает методл для получения городов 
3) метод search с фразой запроса и ключем города 
4) все группы получить сразу нельзя - ограничение максимальное 1000, поэтому 
все запросы делаются в методе API execute - 25 запрососв по 20 групп (чтобы не пергружать вывод на страницу) 
5) когда получены 20 групп вызывается getMemers (с дополнительным ключем sex) - также запрос частотный и можно только 10000 юзеров обработать, также вызывается в методе execurte по 25 запросов 
6) при получении юзеров они обрабатываются и считается процент мудчин и женьшин 
7) По окончанию обработки всех юзеров и по выбранному на UI проценту того или иного пола, принимается рещение об отображении группы на UI
